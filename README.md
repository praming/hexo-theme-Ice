## 使用:

### 评论系统

> 在本主题的`_config.yml`配置文件

#### 来比力

```
enable: true
livere_uid: 你注册的来比力 uid
```

#### gitment
```
  enable: false
  owner: 
  repo: 
  client_id: 
  client_secret: 
  perPage: 10
```

### 友言评论(服务不稳定)
```
  enable: false
  uyan_id: 
```

### Disqus评论
```
  enable: false
  shortname: 
  count: false
```

### 畅言评论(需要ICP备案)
```
  enable: false
  appid: 
  conf: 
```
